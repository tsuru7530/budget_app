# 地区別予算管理アプリ
## 説明
地区別に予算を管理するアプリです。

## URL
https://budget-app-0eh2.onrender.com

## 動機
現在の職場で地区ごとにExcelで予算を管理ており、シートやファイル数が多く面倒なので、一括で管理できるアプリを作りたいと考えた。

## 内容
* 地区情報
  * 登録
  * 編集
  * 削除
  * 検索
* 地区別予算情報
  * 登録
  * 編集
  * 削除
* 予算別執行情報
  * 登録
  * 編集
  * 削除

## 技術
* 言語
  * Ruby on Rails
  * HTML
  * CSS
  * JavaScript
* 機能
  * 画像投稿(active storage)
  * 画像のプレビュー
  * GoogleMapの位置情報保存(GoogleMapsApi)
  * デプロイ(render)
  * 本番環境での画像保存(amazon s3)
  * テスト(rspec)
    * モデル単体テスト
    * コントローラテスト
    * 結合テスト
